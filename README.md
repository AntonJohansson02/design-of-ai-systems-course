# Design of AI Systems Course

A collection of assignments exploring AI systems design, machine learning, and data analysis.

## Assignments

| Assignment | Topic | Description |
|---|---|---|
| **Ass1** | Temperature Prediction | Forecasting daily minimum temperatures using time series data |
| **Ass2** | Recommender Systems | Movie recommendation engine with genre and review analysis |
| **Ass3** | Air Quality Analysis | Pollution analysis and geographic data insights from urban areas |
| **Ass4** | Diagnostic Systems | Medical classification system (breast cancer diagnosis) |
| **Ass5** | Semantic Segmentation | Satellite image segmentation using deep learning |
| **Ass6** | Game-Playing AI | Monte Carlo Tree Search (MCTS) implementation for game strategies |
| **Ass8** | Mini-Project: Housing | Price prediction using feature engineering and ensemble models |

## Technologies

- Python, PyTorch, TensorFlow
- scikit-learn, XGBoost, Gradient Boosting
- pandas, numpy, matplotlib, seaborn
- Jupyter Notebooks

## Structure

Each assignment folder contains:
- Jupyter notebooks with analysis and models
- Supporting datasets (CSV, pkl files)
- Visualizations and results
- Python utility modules
